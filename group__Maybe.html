class yafl::Maybe

template<typename T>
class yafl::Maybe< T >

Maybe class.

Generalization of the Maybe class for any type other than void.

This class implements the Maybe type by realizing concepts from functional programming such as Functor, Applicative and Monad. The implementation for maybe class was based on Haskell Maybe type as defined in https://hackage.haskell.org/package/base-4.18.0.0/docs/Data-Maybe.html
