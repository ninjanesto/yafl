class yafl::Either

template<typename ErrorType, typename ValueType>
class yafl::Either< ErrorType, ValueType >

Either class.

Generalization of the Either class for any error and value types other than void.

This class implements the Either type by realizing concepts from functional programming such as Functor, Applicative and Monad. The implementation for Either class was based on Haskell Either type as defined in https://hackage.haskell.org/package/base-4.18.0.0/docs/Data-Either.html
